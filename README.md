# Pytorch-Based-Bert-Entity-Extraction

## Dataset : [Link to Dataset](https://www.kaggle.com/abhinavwalia95/entity-annotated-corpus) 
This is the extract from GMB corpus which is tagged, annotated and built specifically to train the classifier to predict named entities such as name, location, etc.

### Context
Annotated Corpus for Named Entity Recognition using Groningen Meaning Bank corpus for entity classification with popular features by NLP and ML.
